# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 3.x.y   | ✅         |
| 2.x.y   | ✅         |
| 1.x.y   | ❌         |

## Reporting a Vulnerability

If you find a vulnerability in this crate or one of its dependencies, you can [report it privately](https://github.com/clechasseur/rs-fmt-check/security/advisories/new).
